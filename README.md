# Flying to Trash 🚀🗑️

**Estrutura base**

> _Lista estática com layout e remoção simples_

Um componente React desenvolvido em TypeScript que exibe uma lista interativa de membros de uma equipe e permite remover itens dinamicamente.

## O que foi feito

- Lista de 5 membros da equipe com nome, função e universo
- Avatares representados por círculos coloridos (placeholder)
- Botão `×` remove o membro do estado local com `useState`
- Estilização via inline styles (sem Tailwind ainda)
- Fundo escuro `#0e0e10`, tipografia limpa

## Stack

- React 18 + TypeScript
- Vite 6

## Como rodar

```bash
npm install
npm run dev
```
